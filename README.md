# LightScape Photonics 📸

A modern web application for the **LightScape: Photonics Photography Event** organized by IEEE Photonics SSN. This platform allows participants to submit their photography entries and displays approved submissions in an elegant gallery format.

## 🌟 Features

-   **Submission Portal**: Easy-to-use form for participants to submit their photography entries
-   **Gallery Display**: Beautiful gallery showcasing approved submissions
-   **Responsive Design**: Optimized for both desktop and mobile devices
-   **Real-time Updates**: Dynamic content loading from Firebase database
-   **Interactive Preview**: Click-to-view detailed submission information

## 🚀 Tech Stack

-   **Frontend**: React + TypeScript
-   **Styling**: Tailwind CSS
-   **Build Tool**: Vite
-   **Database**: Firebase
-   **UI Components**: Custom components with shadcn/ui
-   **Package Manager**: pnpm

## 🛠️ Getting Started

### Prerequisites

-   Node.js (v16 or higher)
-   pnpm package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/LightScape_Photonics.git
cd LightScape_Photonics
```

2. Navigate to the client directory:

```bash
cd client
```

3. Install dependencies:

```bash
pnpm install
```

4. Start the development server:

```bash
pnpm dev
```

5. Open your browser and visit `http://localhost:5173`

## 🔧 Available Scripts

-   `pnpm dev` - Start development server
-   `pnpm build` - Build for production
-   `pnpm preview` - Preview production build
-   `pnpm lint` - Run ESLint
